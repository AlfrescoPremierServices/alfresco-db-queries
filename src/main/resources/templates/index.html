Alfresco Database Reporting

  • Large Folders - Large Folders are those folders containing large number of child nodes. Alfresco recommends to keep then number of child nodes per single folder under 1000.
  • Large Transactions - Large Transactions are those transactions containing large number of nodes. Alfresco recommends to keep the number of nodes per single transaction under 10K.
  • Archived Nodes - Trash can should be maintained to keep the number of archived nodes to a minimum.
  • Nodes Disk Space by Mimetype - Number of nodes and total occupation on disk by Mimetype.
  • Nodes by Type - Number of nodes by content type i.e. folder, file, etc.
  • Nodes by Store - Number of nodes by store type i.e. workspace://SpacesStore or archive://SpacesStore.
  • Activities Feed - Number of events by activity type.
  • Lock Resources - List of resources and their status.
  • Users and Groups - Number of users and groups.
  • Workflows - Number of process instances and tasks.
