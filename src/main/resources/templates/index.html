Alfresco Content Services Database Reporting

  • Database Size - Database Size Information
  • Large Folders - Large folders are those folders containing large number of child nodes. Alfresco recommends to keep then number of child nodes per single folder under .
  • Large Transactions - Large transactions are those transactions containing large number of nodes. Alfresco recommends to keep the number of nodes per single transaction under .
  • Access Control List - Number of ACLs and ACEs.
  • Content Model Properties - List of properties by content model.
  • Activities Feed - Number of events by activity type.
  • Activiti Workflows - List of workflows and tasks.
  • Archived Nodes - Trash can should be maintained to keep the number of archived nodes to a minimum.
  • Nodes Grouped by Mimetype - Number of nodes and total occupation on disk by Mimetype.
  • Nodes Grouped by Content Type - Nodes grouped by content type i.e. folder, file, etc.
  • Nodes Grouped by Content Type and Creation Date - Number of nodes by content type grouped by month i.e. folder, file, etc.
  • Nodes by Store - Nodes grouped by store type i.e. workspace://SpacesStore or archive://SpacesStore.
  • Lock Resources - List of resources and their status.
  • Authorities - List Number of users and groups.
  • JMX Properties - JMX properties set in DB.
  • Patches - List of applied patches.
  • Generate Report - Run all queries.
